Institute Technical Summer Project - Agrover

Apr 2022 - Aug 2022

Associated with Indian Institute of Technology, Bombay

Project Overview

Agrover is a project aimed at automating the disease identification and treatment process in agriculture. It is designed to address one of the key issues farmers face: the harmful effects of manually spraying pesticides and insecticides. By creating an autonomous rover equipped with disease identification technologies and automated pesticide spraying, this project seeks to improve farmers' health and productivity.

Problem Statement

Two-thirds of the population in our country is involved in agriculture, but despite advancements in this sector, a cost-effective solution to identifying and treating plant diseases is lacking. Manual spraying of pesticides is harmful to farmers and decreases their quality of life. Agrover aims to solve this problem by automating disease identification and guiding pesticide spraying autonomously.

Technologies Used

  • Python (Machine Learning and Image Processing for Disease Identification)
  • Raspberry Pi for onboard computation
  • Autonomous Navigation System
  • Spraying Mechanism for automated pesticide application

My Role

As part of the team, I was responsible for the integration of the disease identification model using Python, which involved developing machine learning algorithms for identifying various crop diseases based on image data. Additionally, I worked on the design and implementation of the automated pesticide spraying mechanism.
