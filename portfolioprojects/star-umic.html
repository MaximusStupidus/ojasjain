Project Overview

This project was designed to create an autonomous robot capable of navigating various arenas and completing tasks such as ball detection, obstacle avoidance, and navigation. The project demanded proficiency in several technologies, including Python (specifically NumPy and OpenCV libraries), SolidWorks, Robot Operating Systems (Gazebo and RVIZ), and Ubuntu.

Technologies Used

  • Python (NumPy, OpenCV)
  • SolidWorks
  • Robot Operating Systems (Gazebo, RVIZ)
  • Ubuntu

My Role

As part of the development team, I focused on implementing autonomous features such as obstacle avoidance and navigation. I also worked on integrating Python algorithms for ball detection using OpenCV and supported simulation efforts in Gazebo and RVIZ.
