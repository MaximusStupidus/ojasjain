Nifty 50 Stock Screener

Sep 2022 - Jan 2023

Associated with Indian Institute of Technology, Bombay

Project Overview

The Nifty 50 Stock Screener is a modular application that performs Exploratory Data Analysis (EDA) on the top 50 Nifty stocks. It also implements a deep learning model for future price prediction, with an accuracy of 57%. The app calculates various financial parameters such as ROE, ROCE, ROI, and presents them via a frontend interface for easy analysis.

Problem Statement

There is a need for an industry-standard screening software that can help investors analyze stocks and predict future stock prices. Our goal with the Nifty 50 Stock Screener is to create a powerful tool that can not only perform financial analysis but also predict price movements with the help of machine learning algorithms.

Technologies Used

  • Python (for data analysis and machine learning)
  • Pandas, NumPy (for EDA)
  • Logistic Regression (for price prediction)
  • Frontend built using Flask and Bootstrap
  • Deep learning model for future stock price prediction

My Role

I developed the stock screening app and worked on implementing both the logistic regression and deep learning models for price prediction. Additionally, I contributed to the frontend development using Flask, presenting financial data (ROE, ROCE, ROI) to users in a user-friendly format.
